Portility v1.3.0 — Beta Test Install
=====================================

1. Unzip "portility-1.3.0.zip" to a folder on your computer
   (e.g. Desktop/portility)

2. Open Chrome and go to:  chrome://extensions

3. Turn on "Developer mode" (toggle in the top-right corner)

4. Click "Load unpacked"

5. Select the folder you unzipped in step 1

6. Portility should now appear in your extensions toolbar.
   If you don't see it, click the puzzle-piece icon in the
   Chrome toolbar and pin "Portility".

To update later:
- Replace the folder contents with the new version
- Go to chrome://extensions and click the refresh icon on Portility

To uninstall:
- Go to chrome://extensions and click "Remove" on Portility
